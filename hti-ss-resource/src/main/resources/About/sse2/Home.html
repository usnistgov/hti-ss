Supporting

The NIST Syndromic Surveillance Test Suite Edition 2 supports the testing of HL7 v2.5.1 messages in support of the Syndromic Surveillance Community.

  • HL7® Version 2.5.1 Implementation Guide: Syndromic Surveillance, Release 1 - US Realm Standard for Trial Use, July 2019
  • ASTP ONC Health IT Certification – (f)(2) SVAP 2024
ASTP ONC Health IT Certification Start Certification Testing Start Certification Testing

The NIST Syndromic Surveillance Test Suite Edition 2 is the test tool used for certifying ASTP ONC Edition Health IT Modules. The capability used for ASTP ONC Certification can be accessed by navigating to the ASTP ONC Certification Test Plan via the HL7 Context-based option. The ASTP ONC Certification Test Plan is used to certify Modules for the 170.315(f)(2) Transmission to public health agencies - syndromic surveillance ASTP ONC criterion SVAP 2024.

Have a Question?

A Google Group (HL7v2 Syndromic Testing) has been established for discussion/questions about the Implementation Guide, testing tool, and testing issues. No membership is required. A Google account is required for posting.

  • Site: http://groups.google.com/d/forum/hl7v2-syndromic-testing
  • Email: hl7v2-syndromic-testing@googlegroups.com

Note: The HL7v2 Syndromic Testing Google Group supports both Edition 1 (covering the 2015 Implementation Guide Release) and Edition 2 (covering the 2019 Implementation Guide release) of the NIST Syndromic Surveillance Test Suites. When posting a question, please indicated the tool edition and implementation guide release.
