Description

The pediatrician suspects measles and orders a Real-time RT-PCR to confirm, and provides instruction for home isolation to prevent potential spread.

A nurse weighs the patient, measures his length, and charts the data in the patient’s record at 8:35. This information may be used by the physician if ordering the take-home prescription.  At 8:55 AM, the physician assigns the patient the working diagnoses of measles: (ICD-10 CM diagnosis code of B05.9 Measles without complication), Conjunctivitis (ICD-10 CM diagnosis code of H10.10 Acute atopic conjunctivitis, unspecified eye), Fever (ICD-10 CM diagnosis code of R50.9 Fever, unspecified), and Cough (ICD-10 CM diagnosis code of R05.9 Cough, unspecified)  are documented as final in the patient`s electronic medical record. Suspecting Measles, the provider takes a nasal swab to send for Real-time RT-PCR (CPT-4 code of 87798 Infectious agent detection by nucleic acid). The provider recommends isolation at home, providing education to the parents (CPT-4 code of 98960 Education and Training for Patient Self-Management). The visit is closed at 9:05 am after the nurse collects the sample to send to the lab.  

NIST Primary Care is an outpatient facility that routinely sends electronic syndromic surveillance data to the City Health Department (NIST PH SS FAC 1) in accordance with a city regulation. At 6:00 PM, the facility`s electronic health record module for syndromic surveillance data assembles and transmits a Discharge ADT message with height and weight information, as well as the working and final diagnoses for this patient encounter. The patient name and birthdate are de-identified in the submission, so no date of birth is included, and the name is represented with name type 'S' to indicate that the name is empty for anonymity.

  • Diagnoses are submitted test using either ICD-10-CM or SNOMED-CT: Per ASTP/ONC, for certification testing  a Health IT Module can use either the ICD-10-CM or SNOMED CT® value sets in the submitted messages for all of the test steps in the Syndromic Surveillance Test Suite.
